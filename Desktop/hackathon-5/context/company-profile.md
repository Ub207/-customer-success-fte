# TechCorp SaaS - Company Profile

## Company Overview
TechCorp SaaS is a B2B software company providing project management and team collaboration tools for mid-market and enterprise customers (50-5000 employees).

**Founded:** 2018
**Customers:** 2,500+ businesses
**Users:** 150,000+ active users
**Revenue:** $12M ARR
**Support Volume:** ~500 tickets/day

## Products

### TechCorp Core (flagship)
- Task management with AI prioritization
- Team workspaces and channels
- Time tracking and reporting
- Integration marketplace (200+ integrations)
- Pricing: Starter ($9/user/mo), Pro ($19/user/mo), Enterprise (custom)

### TechCorp Analytics
- Real-time project dashboards
- Burndown charts and velocity metrics
- Custom report builder
- Pricing: Add-on at $5/user/mo

### TechCorp API
- REST and GraphQL APIs
- Webhook support
- OAuth 2.0 authentication
- Rate limits: 1000 req/min (Pro), 5000 req/min (Enterprise)

## Support Tiers
- **Free/Starter:** Community forum + email (48hr SLA)
- **Pro:** Email + chat (8hr SLA, business hours)
- **Enterprise:** Email + chat + phone (2hr SLA, 24/7)
